# bookbot

First guided project from [Boot.dev](https://www.boot.dev).

Takes a `.txt` file and reports the word count and character frequency,
sorted from most to least common.

## Usage

```
python3 main.py <path_to_book>
```

## Example

```
============ BOOKBOT ============
Analyzing book found at books/frankenstein.txt...
----------- Word Count ----------
Found 75767 total words
--------- Character Count -------
e: 44538
t: 29493
a: 25894
...
============= END ===============
```
